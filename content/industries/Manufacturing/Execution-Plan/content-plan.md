# Manufacturing Content Plan

## Content Strategy Overview

Two-channel approach:
1. **LinkedIn** (2-3x/week) - Build awareness, credibility, and engagement
2. **Website** (1-2x/month) - SEO, depth, case studies as proof points

Content types:
- **Industry-specific** (manufacturing data challenges, use cases)
- **Generic** (data infrastructure, analytics, AI/automation topics that apply cross-industry)

---

## LinkedIn Content (2-3x per week)

### Content Mix

| Type | Frequency | Purpose |
|------|-----------|---------|
| Insight/observation posts | 1-2x/week | Thought leadership, engagement |
| Case study snippets | 1x/week | Social proof, concrete results |
| Industry commentary | 1x/week | Relevance, timeliness |
| Behind-the-scenes / process | 1x/month | Authenticity, differentiation |

### Manufacturing-Specific Topics

- Data scattered across Salesforce, QuickBooks, inventory systems
- The "hire internally vs. partner" decision
- What unified visibility actually looks like
- Board/investor reporting challenges for growing manufacturers
- Post-acquisition data integration
- Symptoms of outgrown data infrastructure
- New data hire onboarding challenges (how to accelerate them)

### Generic Topics (Cross-Industry)

- Data warehouse basics for mid-market companies
- When to build vs. buy analytics capabilities
- The cost of manual data reconciliation
- Dashboard design principles
- Data quality fundamentals
- AI/automation use cases for SMBs

## Content Creation Process

### LinkedIn Posts
1. Draft post (15-20 min)
2. Review/edit (5 min)
3. Schedule in LinkedIn or scheduling tool
4. Engage with comments same day

### Website Articles
1. Outline (30 min)
2. First draft (1-2 hours)
3. Edit/refine (30 min)
4. Add visuals if needed
5. Publish and share on LinkedIn

### Case Studies
1. Review project notes and deliverables
2. Draft narrative (1-2 hours)
3. Get client approval if named
4. Design/format for website
5. Create LinkedIn snippets from key points
